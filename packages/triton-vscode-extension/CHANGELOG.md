# Changelog

## 0.3.0

- First Marketplace beta / pre-release package for `Triton Architecture Explorer`.
- Opens Triton's browser-based architecture diagram from VS Code-compatible IDEs.
- Supports Scala and sbt workspaces as the initial MVP target.
- Starts the local Triton browser app and runtime when configured.
- Deep-links from the browser diagram back into source files in the IDE.
- Adds runtime-backed refresh, `sbt test`, and coverage actions.
- Includes the new Triton cube icon and browser favicon branding.
